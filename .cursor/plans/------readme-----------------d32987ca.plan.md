# نوشتن README.md جامع برای پروژه

## وضعیت فعلی

- README.md موجود است اما ساده و ناقص است
- پروژه شامل 5 فیچر اصلی است که هر کدام یک فایل پایتون و یک فایل رجیستری دارند
- فایل‌ها در ریشه پروژه قرار دارند و سازماندهی نشده‌اند

## فیچرهای شناسایی شده

1. **تبدیل MP4 به MP3**: `convert_mp4_to_mp3.py` + `convert to mp3.reg`
2. **تبدیل M4A به MP3**: `convert_m4a_to_mp3.py` + `add_right_click_m4a.reg`
3. **تقسیم ویدیو از وسط (MP4)**: `split_middle_overlap.py` + `add_right_click_split_middle_python.reg`
4. **تقسیم ویدیو از وسط (MP3)**: `split_middle_overlap.py` + `add_right_click_split_middle_python_mp3.reg`
5. **اضافه کردن موزیک به MP3**: `add_music.bat` + `add music.reg`
